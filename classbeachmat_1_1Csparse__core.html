Template Parameters
ALTIterator class for the workspace.
Parameters
cThe index of the column to extract.
workA pointer or iterator to the workspace in which the column values are to be stored. This should have at least last - first addressable elements.
firstThe index of the first row of interest.
lastThe index of the first row that is not of interest.
emptyValue corresponding to zero, almost always 0.
Returns
work is filled in with the contents of column c from rows [first, last). If no non-zero element exists, the corresponding entry of work is set to empty.
