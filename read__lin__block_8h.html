std::unique_ptr<lin_sparse_matrix> beachmat::promote_to_sparse ( std::unique_ptr< lin_matrix > &  ptr)
inline

Promote an existing pointer to a lin_matrix into a pointer to a lin_sparse_matrix. This allows the code to switch between dense/sparse processing based on the observed class.

Parameters
ptrA pointer to a lin_matrix, typically produced by read_lin_block(). Note that this will no longer be valid upon successful return of this function.
Returns
A pointer to a lin_sparse_matrix instance. This function will throw an error if promotion cannot be achieved.

std::unique_ptr<lin_matrix> beachmat::read_lin_block ( Rcpp::RObject  block)
inline

Read a logical, integer or numeric block into an instance of a lin_matrix subclass. This can then be used to perform class- and type-agnostic extraction of row/column vectors.

Parameters
blockAn R object containing an ordinary submatrix, dgCMatrix, lgCMatrix or SparseArraySeed.
Returns
A pointer to a lin_matrix instance. This function will automatically choose the most appropriate subclass or throw an error if none are available.
