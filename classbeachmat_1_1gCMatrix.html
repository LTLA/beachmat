Detailed Description

template<class V, typename TIT>
class beachmat::gCMatrix< V, TIT >

Sparse logical or numeric matrices in the lgCMatrix or dgCMatrix format, respectively, from the Matrix package.

It is unlikely that this class will be constructed directly by users; most applications will use read_lin_block() or read_lin_sparse_block() instead.

Template Parameters
VThe class of the Rcpp::Vector holding the R-level data for non-zero values.

Parameters

Returns
