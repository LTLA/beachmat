Detailed Description

template<typename TIT, typename I>
struct beachmat::sparse_index< TIT, I >

Sparse index container, holding the number of non-zero elements extracted from a single row/column along with pointers to arrays containing their indices and values.

For extracted rows, the indices refer to column positions; for extracted columns, the indices refer to the row positions. In both cases, indices can be assumed to be zero-indexed and sorted.

Template Parameters
TITThe type of the (const) random-access iterator pointing to the data values.
IThe integer type of the index.
