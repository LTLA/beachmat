Detailed Description

template<class V>
class beachmat::lin_ordinary_matrix< V >

Logical, integer or numeric matrices in the "ordinary" R format, i.e., column-major dense arrays.

It is unlikely that this class will be constructed directly by users; most applications will use read_lin_block() instead.

Template Parameters
VThe class of the Rcpp::Vector holding the R-level data.

Parameters

Returns
A pointer is returned to the values of c as doubles, starting at the first element. This may or may not involve populating work with values copied from the underlying matrix; a copy will have been performed iff the return value compares equal to work.
