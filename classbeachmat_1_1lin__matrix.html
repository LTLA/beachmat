Detailed Description

Virtual base class for a logical, integer or numeric (i.e., double-precision) matrix, providing methods to extract rows or columns in dense form.

Extract a column as an array of doubles.

Parameters
cIndex of the column of interest.
workThe workspace, potentially used to store extracted values. This should have at least last - first addressable elements.
Returns
A pointer is returned to the values of c as doubles, starting at the first element. This may or may not involve populating work with values copied from the underlying matrix; a copy will have been performed if the return value compares equal to work.
