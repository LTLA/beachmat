Detailed Description

template<class V>
class beachmat::ordinary_reader< V >

Type-agnostic reader for row/column data from an ordinary R matrix.

Unlike the lin_ordinary_matrix class template, this is type-agnostic.

Note
This is an internal class and should not be constructed directly by beachmat users.
Template Parameters
VAn Rcpp::Vector class, used to contain the contents of the matrix.
