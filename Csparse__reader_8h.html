template<typename OUT , typename TIT , typename ALT , typename I >
sparse_index<OUT, int> beachmat::transplant ( sparse_index< TIT, I >  ref,
ALT  work_x,
I *  work_i 
)
inline

Transplant indices and values into their respective workspaces and use them to construct a new sparse_index object. This is necessary for type conversions between the stored and expected types of non-zero values.

Note
This is an internal function and should not be called directly by beachmat users.
Template Parameters
OUTIterator to be stored in the new sparse_index. This should be the const-type counterpart to ALT.
TITconst iterator of native type, i.e., equal to the original data values in the matrix.
ALTNon-const iterator to the workspace of the desired type.
IInteger type of the index.
Parameters
refsparse_index containing pointers from a column (usually) of a sparse representation.
work_xPointer to the workspace for the data values, usually of a different type to that of the native representation. This should have space for at least ref.n values.
work_iPointeger to the workspace for the indices of the non-zero values. This should have space for at least ref.n values.
Returns
A sparse_index containing iterators for work_x and work_i.
